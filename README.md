# Iris Classification & Machine Learning Exercises

This repository contains a series of Python implementations for fundamental machine learning tasks, focusing on data preprocessing, classification algorithms, and performance evaluation.

## 🚀 Projects Overview

The scripts cover the following machine learning concepts:

### 1. Data Partitioning
* **80/20 Split:** Basic train-test partitioning.
* **10-Fold Cross-Validation:** Robust data splitting.
* **Stratified 10-Fold CV:** Maintaining class balance across folds.

### 2. Classification Algorithms
* **Decision Trees:** Implementations using both **Entropy (Information Gain)** and **Gini Index**.
* **Naïve Bayes:** * **CategoricalNB** applied to the "Buys Computer" dataset.
  * **GaussianNB** applied to the Iris dataset.
* **k-Nearest Neighbors (k-NN):** * Comparison between **Euclidean** and **Manhattan** distance metrics.
  * Hyperparameter tuning ($k=5$ and $k=7$).

### 3. Evaluation Metrics
* **Classification Metrics:** Confusion Matrix, Precision, Recall, F1-Score, and Accuracy.
* **Regression Metrics:** Mean Absolute Error (MAE), Mean Squared Error (MSE), and Root Mean Squared Error (RMSE).
* **Visualizations:** ROC Curves, AUC calculation, and Feature Correlation Heatmaps.

### 4. Feature Selection
* **Information Gain Ratio:** Identifying the top 2 features in the Iris dataset for optimal classification.
